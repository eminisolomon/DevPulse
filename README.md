# DevPulse

DevPulse is a comprehensive mobile application built with React Native and Expo that empowers developers to gain deep insights into their coding habits. By integrating with the WakaTime API, DevPulse provides a beautiful, data-rich dashboard to track development progress, visualize productivity trends, and manage goals.

[![Get it on Google Play](https://play.google.com/intl/en_us/badges/static/images/badges/en_badge_web_generic.png)](https://play.google.com/store/apps/details?id=com.techx.devpulse)

[![Download on the App Store](https://developer.apple.com/app-store/marketing/guidelines/images/badge-download-on-the-app-store.svg)](https://apps.apple.com/app/id1665262626)

## Features

- **Personalized Dashboard**: View your daily coding activity, top languages, and project breakdowns at a glance.
- **Detailed Analytics**: Dive deep into your statistics with interactive charts and historical data comparisons.
- **Project Tracking**: Monitor specific projects to see where your time is going.
- **Goal Management**: Set and track coding goals to stay motivated and productive.
- **Leaderboards**: See how you stack up against other developers globally or within your organization.
- **Customizable Themes**: Choose from a variety of themes and accent colors to personalize your experience.

## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/eminisolomon/DevPulse.git
   cd DevPulse
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the app:**

   ```bash
   npm run start
   ```
